About Me

Hello, I'm PineappleDoge, also known as PineappleDoge1 on Roblox! I'm currently a high-school student planning to major in art/animation, alongside programming as a hobby. I'm offering my services as a Programmer, Animator, Rigger, & UI Designer.

I specialize in creating high-quality, highly-adjustable, well documented systems using Luau, as well as using a variety of other software to create beautiful, robust products.

What I Use

As far as programming tools go, I currently work using Knit, a framework/library created by Sleitnick by that allows me to work at a quick pace, and maintain a beautiful codebase. I program primarily inside Roblox Studio's code editor, although I'm capable of using Rojo and Sublime Text for clients that prefer version control.

As far as designing terrain, animations, and UI, I prefer to conceptualize and do research using external programs, then assemble and polish the product inside studio.

Animation | Rigging

  • Blender - I use Blender for designing animations. It's easy to use, and allows me to create higher quality work, while maintaining compatability.
  • SyncSketch - Mainly used for animation analysis, allows me to plug in mp4s and analyze keyposes, extremes, and in-betweens.
  • RigEdit Plus - Used for non mesh-deforming rigs.

Scripting

  • Github - Version control is a major lifesaver, and allows me to sync work with clients/other workstations.
  • Drawio - My flowcharting tool, allows me to plan logic in advance, and provide the client with an idea of how the system will look in-action before programming.
  • Sublime Text - Awesome text editor, wrote this portfolio using it, enough said about it.

UI Design

  • Lunacy - Allows me to design high-quality layouts using intutitve tools
  • MSPaint - Quick sketches for very basic conceptualization.
  • Medibang Paint Pro - Used for icons, or more hand-drawn styles.

Other

  • Roblox Studio - Obvious, but used for assembling and importing end-products if desired

Pricing Details

My prices are negotiable! Due to the variety of products I'm able to create, the prices can vary drastically. If you're questioning a price about commissioning me, the best course of action is to contact me through my email, Discord, or by Twitter. Although complexity does play a major part in what I charge, below is some details about my minimum rates.

I accept payments in USD ($) and Robux (R$). If the latter method (R$) is chosen, group funds are heavily preferred, and the funds should already be deposited in the group to reduce wait-time.
